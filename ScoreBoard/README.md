# ScoreBoard

학생의 이름과 시험 점수를 관리하는 성적표 클래스를 구현합니다.

`ScoreBoard` 클래스를 작성하세요:

**필드**

- `_names`: 학생 이름을 저장하는 문자열 배열
- `_scores`: 점수를 저장하는 정수 배열
- `_count`: 등록된 학생 수

**생성자**

- 용량(capacity)을 매개변수로 받아 배열을 초기화함

**속성**

- `Count`: 등록된 학생 수를 반환함 (읽기 전용)

**메서드**

- `Register(string name, int score)`: 학생을 등록함. 용량 초과 시 "성적표가 가득 찼습니다."를 출력함

**인덱서**

- `string this[int index]` (get): 번호로 학생 이름을 조회함. 범위를 벗어나면 빈 문자열("")을 반환함
- `int this[string name]` (get/set): 이름으로 점수를 조회하거나 수정함. 배열을 순회하여 이름이 일치하는 항목의 점수를 반환하고, 없으면 -1을 반환함

**Main 함수에서 테스트**

1. 용량이 5인 `ScoreBoard` 생성
2. "김민수"(85점), "이지영"(92점), "박서준"(78점) 등록
3. 등록된 학생 수 출력
4. 번호로 이름 조회 (반복문 사용)
5. 이름으로 점수 조회 ("김민수", "이지영", "홍길동")
6. "김민수" 점수를 95점으로 수정 후 다시 조회

## 예상 실행 결과

```
등록된 학생 수: 3명

0번: 김민수
1번: 이지영
2번: 박서준

김민수 점수: 85점
이지영 점수: 92점
홍길동 점수: -1점

김민수 수정된 점수: 95점
```
