# FruitPriceList

과일 이름과 가격을 관리하는 가격표 클래스를 구현합니다.

`FruitPriceList` 클래스를 작성하세요:

**필드**

- `_names`: 과일 이름을 저장하는 문자열 배열
- `_prices`: 가격을 저장하는 정수 배열
- `_count`: 등록된 과일 수

**생성자**

- 용량(capacity)을 매개변수로 받아 배열을 초기화하고 `_count`를 0으로 설정함

**속성**

- `Count`: 등록된 과일 수를 반환함 (읽기 전용)

**메서드**

- `Add(string name, int price)`: 과일을 추가함. 용량 초과 시 "가격표가 가득 찼습니다."를 출력함

**인덱서**

- `int this[string name]` (get): 이름으로 가격을 조회함. 배열을 순회하여 이름이 일치하는 항목의 가격을 반환하고, 없으면 -1을 반환함
- `string this[int index]` (get): 순서로 과일 이름을 조회함. 범위를 벗어나면 빈 문자열("")을 반환함

**Main 함수에서 테스트**

1. 용량이 5인 `FruitPriceList` 생성
2. "사과"(1500원), "바나나"(3000원), "딸기"(8000원) 추가
3. 등록된 과일 수 출력
4. 이름으로 가격 조회 ("사과", "바나나", "포도")
5. 순서로 과일 이름 조회 (반복문 사용)

## 예상 실행 결과

```
등록된 과일 수: 3개

사과 가격: 1500원
바나나 가격: 3000원
포도 가격: -1원

0번: 사과
1번: 바나나
2번: 딸기
```
